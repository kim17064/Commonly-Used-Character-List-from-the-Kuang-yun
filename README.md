# 廣韻常用字表（作詩用）

收錄廣韻中之常用韻字，方便作詩押韻。

##  為何用廣韻

平水韻中四聲之韻部劃分不齊。且迥拯同用，徑證同用。可見其重實用而輕語音。

##  整理原則

1. 照廣韻之同用獨用規則。

2. 四聲之韻部劃分對齊。

3. 廣韻中四聲之韻部劃分不同時，採分類較多之劃分。

具體為：文獨用，欣獨用。鹽添同用，咸銜同用，嚴凡同用。上去入三聲類推。

與平水韻之異：除上述外，迥不與拯等同用，徑不與證嶝同用。去聲廢獨用。
